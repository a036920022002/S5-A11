# Alpha Camp Semester 2-3 S5-A11 Assignment

學期 2-3 短網址產生器

### 開啟方式

- 於終端機執行程式碼

```js
 git clone https://github.com/a036920022002/S5-A11
```

- 於資料夾中增加.env 檔
  並新增以下環境參數

```js
My_ENV= "mongodb+srv://alpha:camp@cluster0.fye8rmm.mongodb.net/shortURL?retryWrites=true&w=majority"`
```

- 於終端機執行
  `npm i express`
  及
  `npm i dotenv`

- 執行腳本
  `npm run dev`

- 進入主畫面
